round_table
===========

App for managing round tables (google hangouts).

```bash
$ git clone https://github.com/bbfsdev/round_table.git
$ cd round_table
```

# Backend

## Redis:

```bash
$ sudo apt-get install redis
```

## Sinatra

```bash
$ cd backend
$ bundle install
$ ruby backend.rb
```

# Frontend

```bash
$ cd frontend
```

Install Gulp and Bower globally:

```bash
$ sudo npm install -g gulp bower
```
